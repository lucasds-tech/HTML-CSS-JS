  9. Links Internos

    10. Quando eu colocar dentro de 'href=”../”’ significa que eu vou puxar dados da pasta anterior. ( “./” é a mesma pasta)

    'rel=“prev”' é igual a previous, ir para aba anterior.

    'rel=“next”' é igual a próximo, ir para a próxima aba.

    'target= “_self”' é designado para ela mesma (para a própria pagina/aba)

    'rel=”nofollow”' significa não siga, mas tem relevância mesmo apenas para o mecanismo.

  10. Links e Âncoras

    11. Para acessar meu perfil no linkedin,Clique Aqui. 

    a href=”(link aqui)” target=”_blank” rel=”external”
    • O 'a href' é uma âncora de referência onde você coloca o seu link para acesso. (está primeira tag foi criada através de envelopamento que tem o comando de “Ctrl+shift+p >> enter >> a”)

    • O 'target=“_blank”' simbolizada que você está indicando ao site que este é um link externo, que é pra ser aberto em uma nova aba de internet. Ou seja, link de um outro site e não do mesmo.

    • O 'rel=”external”' é apenas para simbolizar pro site que este é um link externo.
